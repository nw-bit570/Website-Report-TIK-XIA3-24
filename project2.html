Project 02

SnapOutfit

Belanja Cepat, Tampil Hebat. 🛍️

📝 Deskripsi Project

SnapOutfit adalah sebuah toko pakaian daring (e-commerce) yang dirancang untuk menjadi platform penjualan digital yang mandiri. Tujuan utamanya adalah memperluas jangkauan pasar melampaui batas toko fisik dengan menyediakan ruang belanja virtual yang dapat diakses kapan saja. Pengunjung dapat menjelajahi katalog pakaian dan aksesori, menambahkannya ke keranjang belanja, lalu melanjutkan ke proses pembayaran — semuanya dalam satu alur yang ringkas. Dengan memotong jalur perantara penjualan, platform ini berpotensi menjangkau lebih banyak pelanggan secara langsung sekaligus memaksimalkan kenyamanan berbelanja.

⚙️ Teknologi yang Digunakan

React JS
Membangun antarmuka berbasis komponen modular seperti kartu produk, navbar, dan halaman checkout, dengan manajemen data global lewat React Context (AuthContext & CartContext).
Supabase
Layanan back end untuk basis data dan autentikasi. Keamanan privasi data ditanamkan langsung pada basis data menggunakan aturan Row Level Security.
CSS3
Estetika modern minimalis dengan palet netral, sistem grid adaptif yang berubah dari 1 kolom di ponsel menjadi beberapa kolom di desktop.
JavaScript
Logika keranjang belanja yang persisten, pengendali tampilan, serta sinkronisasi data antara peramban dan basis data.

✨ Fitur Utama

🔍

Pencarian Instan

Cari dan saring produk dengan cepat sesuai kebutuhan pengunjung.

🛒

Keranjang Persisten

Isi keranjang tetap tersimpan meski halaman ditutup dan dibuka kembali.

🔐

Autentikasi Aman

Sistem masuk dengan privasi data yang ketat melalui Row Level Security.

🛠️ Catatan Pengembangan

Bagian paling menantang sekaligus paling menarik dari SnapOutfit adalah menjembatani antarmuka pengguna dengan infrastruktur back end Supabase. Alih-alih hanya mengandalkan kode di sisi aplikasi, keamanan data benar-benar ditanamkan langsung pada basis data lewat aturan Row Level Security. Merancang logika keranjang belanja agar tetap konsisten di berbagai perangkat juga memberi pembelajaran berharga tentang pengelolaan state global menggunakan React Context serta arsitektur data relasional yang kokoh.
